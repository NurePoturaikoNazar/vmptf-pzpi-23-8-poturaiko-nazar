# Лабораторна робота 1

Номер в списку 13, кількість завдань 9. 13%9 = 4.

1.	Розробіть сторінку зі списком категорій та виведенням кількості статей у кожній категорії.
2. 	Реалізуйте можливість сортування статей за датою публікації та кількістю коментарів.
3.	Створіть шаблон для відображення кожного коментаря на сторінці статті. Додайте можливість видалення коментарів.
4.	Розробіть систему авторизації та ролей користувачів. Забезпечте редагування та видалення статей лише автором або адміністратором.
